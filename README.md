# WhatIsMyIP

## Overview

### What is WhatIsMyIP?
WhatIsMyIP is a lightweight IP address lookup service that quickly retrieves your public IP address. 

### How to use WhatIsMyIP?
```json
{
  "mcpServers": {
    "whatismyip": {
      "command": "npx",
      "args": [
        "-y",
        "whatismyip-mcp"
      ]
    },
  }
}
```
